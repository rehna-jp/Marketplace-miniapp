# GameMiniLayout

**Type**: component

GameMiniLayout - Layout for game-based mini apps

Features:

- Static header (not floating, part of layout flow)
- Full-height layout (no scrolling)
- Built-in Tabs from

## JSX Usage

```jsx
import { GameMiniLayout } from "@/neynar-farcaster-sdk/game";

<GameMiniLayout tabs={[]} defaultTab="value" />;
```

## Component Props

### tabs

- **Type**: `GameTab[]`
- **Required**: Yes
- **Description**: No description available

### defaultTab

- **Type**: `string`
- **Required**: No
- **Description**: No description available
